# EGORulerView
iOS 带单位的尺子，直尺，基于UICollectionView实现，轻量级，可高度自定义风格。
<br>
# 功能
支持刻度值为小数<br>
支持可选最大值最小值为小数<br>
支持显示的最大值最小值为小数，且不会显示多余的刻度<br>
支持一个完整的长刻度不仅仅是10个<br>
支持自动定位到滑动停止时最近的刻度<br>
支持重用，内存占用低<br>
支持单位，以及能自定义常见样式<br>
...<br>

某些值取一些非常规的值时，可能会有bug，尚未测试。<br>
